# 📰 Fake News Detection using NLP

## 📌 Project Description
This project detects whether a news article is REAL or FAKE using Natural Language Processing and Machine Learning.

---

## 🚀 Features
- Predicts fake or real news
- Simple Streamlit web app
- Fast and accurate model
- User-friendly interface

---

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn

---

## 🤖 Machine Learning Model
- TF-IDF Vectorizer
- Passive Aggressive Classifier

---

## ▶️ How to Run

### 1. Install requirements
pip install -r requirements.txt

### 2. Train model
python model_training.py

### 3. Run app
streamlit run app.py

---

## 📂 Project Structure
- app.py
- model_training.py
- model.pkl
- vectorizer.pkl
- requirements.txt

---

## 🎯 Output
User enters news text → Model predicts REAL or FAKE

---

## 👨‍💻 Conclusion
This project demonstrates how NLP can be used to detect fake news effectively.
